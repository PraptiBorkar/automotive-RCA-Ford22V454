# Root Cause Analysis — Ford Recall 22V454

## Overview
A materials and quality engineering case study analysing 
the Ford front subframe bolt strip-out failure (NHTSA 
Safety Recall 22V454, 2022).

## Tools Applied
- Ishikawa Cause-and-Effect (Fishbone) Diagram
- 5-Why Root Cause Analysis
- 8D Corrective Action Framework
- IATF 16949 Quality Management Standards

## Key Findings
- HSLA steel yield strength degraded from 260 MPa to 
  58 MPa due to accidental annealing
- Root cause: unapproved thermal e-coat stripping at 
  sub-tier supplier
- Systemic gap: OEM audit coverage did not extend to 
  Tier-2 suppliers
